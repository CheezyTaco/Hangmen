# Hangmen 
Multiplayer Hangmen game using multi-threading and TCP  

# Running the Game
    Start the server.py file in python_server.
        Navigate to the python_server file
        Run command: Python3 server.py
    Connect to the server by running game.py.
        Navigate to the Hangmen directory
        Run command: Python3 game.py [Server IP Address]
            example command: Python3 game.py 143.28.212.161
